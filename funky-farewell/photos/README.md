# 📸 How to Add Your Photos

Place your department photos in this folder with these exact filenames:

| File | Role |
|------|------|
| `photo1.jpg` | Large feature photo (Group Photo) |
| `photo2.jpg` | Moment photo |
| `photo3.jpg` | Moment photo |
| `photo4.jpg` | Moment photo |
| `photo5.jpg` | Moment photo |
| `photo6.jpg` | Moment photo |
| `photo7.jpg` | Moment photo |
| `photo8.jpg` | Moment photo |
| `photo9.jpg` | Large feature photo (Department Photo) |

**Supported formats:** `.jpg`, `.jpeg`, `.png`, `.webp`

Until a photo is added, that slot will show an animated placeholder.
Once added, click any photo to open the full lightbox viewer (use ← → arrow keys or the buttons to navigate).
